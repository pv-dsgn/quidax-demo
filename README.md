# quidax-demo

Quidax design system — brand tokens, typography (Uncut Sans Variable), spacing, and Figma MCP integration rules.

See **[CLAUDE.md](./CLAUDE.md)** for the full design system and Figma implementation guide.

## Structure

```
src/styles/
  tokens.css      # Colors, type scale, spacing & gap tokens
  fonts.css       # Uncut Sans Variable @font-face
  typography.css  # Semantic .text-* classes
public/fonts/     # Add uncut-sans-variable.woff2 (licensed)
```
